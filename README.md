# Product Strategy Coach

This folder contains a Codex skill for a **10–15 minute AI product strategy drill**: one realistic multiple-choice scenario, then grading + coaching.

It can take a while to pose a question because it reads and synthesizes the local `.txt` strategy docs. The point is to assimilate that AI Product Strategy content into faster product decisions.

You can practice alone, or you can start the day with your team by solving one problem per day; the idea is a "brain workout" using AI Product Strategy.

## Quickstart

1) Use `SKILL.md` in your Codex setup.
2) In chat, type `GO` (or `GO ECON`, `GO MOAT`, `GO DIFF`, `GO TRUST`, `GO PRICING`, `GO ORG`).
3) Answer with `A`, `B`, `C`, `D`, or `E`; the coach grades and explains.

### Modes (optional)

Use `GO` to let the coach pick a mode, or `GO <MODE>` to force it (`ECON`, `MOAT`, `DIFF`, `TRUST`, `PRICING`, `ORG`). If you don’t force a mode, the coach rotates and avoids repeating the same mode twice in a row.
